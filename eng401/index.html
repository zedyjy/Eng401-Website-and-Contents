My Eng 401 project topic:

A Proposal to Investigate Technical Solutions to Prevent Tiktok from Sharing User Personal Data
Tiktok is known for collecting significant amounts of user personal information. The pilot study area is on how to stop Tiktok from sharing user personal information.
Solutions:
• Storing personal data Tiktok has collected from its users in blockchains instead of databases.
• Using fingerprint hashing on collected data.
• Using AI control technology over data sets.
